Para usar nossa plataforma no ZABBIX:

                                    Para usar o envio com imagens visite nosso GIT:
                                    

                                    https://github.com/MarreraTech/Zabbix
                                    


                                    Criar Script whatsapp.sh para enviar como TEXTO:
                                    nano /usr/lib/zabbix/alertscripts/whatsapp.sh
                                    
                                    Colar o script:
                                    
                                        
                                            #!/bin/bash
                    
                                            SENDTO=$1
                                            SUBJECT=$2
                                            MESSAGE=$3
                                            TOKEN="SEU TOKEN"
                    
                                            parameters=$(cat <<EOF
                                            {
                                                "receiver": "$SENDTO",
                                                "message": {
                                                    "text": "$SUBJECT \\n$MESSAGE"
                                                }
                                            }
                                            EOF
                                            )
                                            url="https://apizap.marrera.net/rest/sendGroup/text/?id=$TOKEN"
                    
                                            # Realização da requisição usando curl e armazenamento da resposta
                                            response=$(curl -s -X POST -H "Content-Type: application/json" -d "$parameters" "$url")
                                        
                                    
                    
                                    Dar as permissões ao script whatsapp.sh
                                    
                                        
                                            chmod +x /usr/lib/zabbix/alertscripts/whatsapp.sh
                                            ln -s /usr/lib/zabbix/alertscripts/whatsapp.sh /usr/bin/whatsapp.sh
                                            chown zabbix. -R /usr/lib/zabbix/alertscripts
                                        
                                    
                                    Em casos de dúvidas, contate o suporte para auxílio referente à integração do Módulo Whatsapp.
                                    Não produzimos script de notificação de nenhuma forma, somente a integração do módulo. O script de alertas e etc é de responsabilidade do usuário!
